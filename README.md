# Beautify
Beautify is a flutter e-commerce app for displaying makeup products ,managing user's orders and sales.


## State Management
- Bloc
- Getx
- Provider

## Design Pattern
-️ MVVM

## Features
- Multiple Themes 
- data caching 
- ️image pickers 
- cart 
- search functionality 
- share products 
- favourites list 
- authentication 
- add and change address 
- keep track of your orders
- Animations 
- Rate products
- Checkout
